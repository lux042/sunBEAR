# CIA FOIA Scraper

A native macOS application for searching the CIA Freedom of Information Act
Electronic Reading Room, collecting document metadata, downloading available
PDFs, and exporting records in an EndNote-compatible tab-delimited format.

## Planned Features

- Advanced CIA Reading Room search
- Multiple collection selection
- Search by title, document number, classification, date, content type,
  and case number
- Automatic pagination through search results
- Metadata extraction from document pages
- PDF downloading
- Embedded PDF text extraction
- EndNote-compatible TSV export
- Preservation-focused metadata export
- Pausable and resumable scraping jobs

## Platform

- macOS
- Swift
- SwiftUI
- SwiftData
- URLSession
- PDFKit

## Project Status

Initial design and research stage.
