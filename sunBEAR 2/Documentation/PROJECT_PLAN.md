# Project Plan

## Phase 1: Search Inspection

- Identify the CIA advanced-search request method
- Record all form parameter names
- Record collection option values
- Record publication-date operator values
- Identify pagination behavior
- Identify search result link structure

## Phase 2: Minimum Working Scraper

- Submit a keyword search
- Parse one results page
- Extract document detail links
- Parse metadata from one document
- Find and download one PDF
- Export one TSV record

## Phase 3: Native macOS Interface

- SwiftUI search form
- Searchable multi-selection collection picker
- Search progress display
- Results table
- Document inspector
- Destination-folder selection

## Phase 4: Full Scraping Jobs

- Pagination
- Rate limiting
- Request retries
- Pause and cancel
- Duplicate detection
- Resume interrupted jobs
- Failure reporting

## Phase 5: Export and Research Tools

- Preservation TSV
- EndNote-compatible TSV
- JSON manifest
- PDF text extraction
- Optional OCR
